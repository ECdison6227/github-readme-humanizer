# 作业帮错题本裁剪器

> 注意：以下故事仅为示例，用于演示“人味 README”的写法，并非本仓库作者的真实经历。

我的一位初中数学老师朋友，每次从作业帮导出错题本 PDF，都要把每道题单独截图出来，拼成给学生上课用的讲义。一份 15 题的卷子，她要切半小时。

这个脚本就是为了让她能直接导出成 15 张独立的题目图片。

## 能做什么 / 不能做什么

| 能做什么 | 不能做什么 |
|----------|------------|
| 自动去掉作业帮顶部 logo 和底部页码 | 不支持扫描版 PDF |
| 把跨页的题目拼成一张图 | 多栏排版没处理 |
| 输出 PNG，按题号命名 | 手写批注识别不了 |

## 快速开始

```bash
pip install -r requirements.txt
python crop_questions.py examples/输入/数学讲义-20260616.pdf
```

输出在 `examples/效果对比/`：

```
题目_01.png
题目_02.png
...
题目_15.png
```

## 一个真实案例

第 4 题题号在第 2 页底部、正文在第 3 页顶部。脚本会判断下一页第一个题号离页眉的距离，如果很远就认为是上一题的续页，自动拼成一张图。

## 踩坑记录

### 第一版：每道题被拆成标题和正文两块

**现象**：15 题切出了 30 张图。
**原因**：`get_text("blocks")` 把“第 1 题”和正文分成两个 block。
**修复**：改成 `get_text("dict")` 的 line/span 级别，把同一行的 span 合并后再匹配。
