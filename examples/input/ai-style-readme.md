# PDF Processor

## Introduction

This project is a powerful and efficient PDF processing tool designed to help users intelligently recognize and crop questions from PDF documents.

## Features

- Intelligent recognition of multiple question formats
- Efficient cropping and output
- Flexible configuration options
- Lightweight and easy to use

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python crop_questions.py input.pdf
```
